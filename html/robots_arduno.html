Arduino

The Arduino is a microcontroller that can easily be programed by a computer via the USB port.  It uses a variant of C and has a huge following so sample code is easy to come by.  I have used it in my rover along with a Raspberry Pi board running Linux.  The Arduino comes in many flavors and not all are created equal  as I have found out.

My first rover used the Uno that has 14 basic I/O pins some analog some digital.  The 6 analog pins have built in A/D converters with 10bits (0-1024) and also can bu used as pulse width modulation for output.  you can hook a verity of devices directly to the Arduino including standard hobby servos, switches, sensors and potentiometers.  There are also a wide verity of shields that just plug into the Ardunio and come with pre built libraries to get you off and running fast.

Next I used the Due which has many more I/O pins and found that I needed to modify the my hardware to be compatible as the timer used for the pulse width modulation for pins 9 and 19 changed form an 16bit timer to a 8bit timer.  To fix this I needed to use pins 11 and 12 that have a 16bit timer.
