Basic Python Programming

Python has many modules and frameworks available. The following is a good basic set of tools.
I have included sample code on GitHub and/or documentation for each module:

Flask: A small lightweight framework for creating APIs and web apps
Request: Python module for handling web services and REST interactions
Multiprocessing: The multi-processing module is useful for parallel task that take a long time. This starts another process on the system
Threading: In many cases simple threading can greatly increase performance when the sub task are not that intensive.
SQLAlchemy: Recommend fast SQL module. Has pagination, migration and many other features
Virtualenv: A python module for setting up standalone projects with python modules in the project dir.
Json: for encoding, decoding and working with json data
Pprint: for printing dictionaries, arrays, and json data generically
