Linux Distributions

Linux is an operating system designed to be very close to UNIX but as a free open source distribution.  Linux in most cases is free of charge as it is open source and tends to be more secure then Windows. It comes in many flavors the main difference is the tools used for administration and instillation. All distributions are based on GNU tools and the Linux Kernel.  Most of the servers on the internet run Linux. 

The most common distributions are Red Hat, Debian and SUSE.  Everyone has an opinion as to what the best distribution is I have gone over some of the most popular here.  All the mainstream distributions of Linux are great and have their own advantages, you can check them out at Linux.org.

Redhat Enterprise is the most common Linux used commercially as it is easy to install and manage, as well as having support from Redhat.  CentOs is the same as Red Hat Enterprise expect for the name and not having Redhat support.  The binary's are the same as it is built from the same distribution as Red Hat Enterprise and it is Free.  Redhat also has a free version called Fedora witch is great but not exactly the same as Redhat's enterprise version.  This server is running CentOS on VMware ESXi.

Debian is also a very common version of Linux and many commercial servers are running it.  In the past it was a little harder to install but the newer versions are easy to install.  Ubuntu is also a very popular Linux version that is based on the Debian distribution.

SuSe is another very popular Linux distribution and is worth checking out.

FreeBSD while not Linux is a very popular open source version of UNIX. It is derived from BSD (Berkeley Software Distribution), the version of UNIX developed at Berkeley.  FreeBSD in some cases is much closer to UNIX then Linux as the original code was based on AT&Ts UNIX.  It is used on many Internet servers and appliances. You can download and try FreeBSD at FreeBSD.org

cygwin is not Linux but is a port of many of the Linux tools for windows.  I love cygwin as it allows me to have access to most of the GNU tools in windows.  For the most part it gives you a Linux shell on windows.
